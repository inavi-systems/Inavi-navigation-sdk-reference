OnNaviInitializeListener

interface OnNaviInitializeListener

An interface for receiving status updates during SDK initialization.

This interface delivers events occurring during the SDK initialization process via callbacks. The initialization proceeds in the following steps:

  1. Map initialization complete - onMapReady is called.

  2. All procedures finished (permission requests, SDK authentication, map data download, etc.) - onInitSuccess is called.

  3. Error occurs during initialization - onInitFail is called.
