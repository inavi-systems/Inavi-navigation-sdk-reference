NaviController

object NaviController : KoinComponent

KMP SDK의 모든 기능을 사용하기 위한 진입점(Entry Point) 역할을 하는 싱글톤 객체입니다.

SDK 초기화부터 검색, 경로, 안내 등 모든 핵심 기능은 NaviController를 통해 호출해야 합니다.
