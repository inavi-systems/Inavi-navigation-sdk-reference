BottomViewEventListener

interface BottomViewEventListener

주행 컴포넌트 BottomView에서 발생하는 이벤트를 3rd Party 앱에서 직접 처리하기 위해 BottomBtnEvent를 받는 인터페이스 입니다.

기본적으로 BottomView에는 자체 동작이 구현되어 있지만, 이 인터페이스를 구현하여 세팅한다면 3rd Party 앱에서 원하는 커스텀 동작으로 대체할 수 있습니다.
